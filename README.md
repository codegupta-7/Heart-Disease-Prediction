# Heart Disease Prediction Using Machine Learning

## Project Overview
This project predicts whether a patient has heart disease using machine learning algorithms.

## Dataset
Heart Disease Dataset

## Algorithms Used
1. Logistic Regression
2. Random Forest
3. K-Nearest Neighbors (KNN)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Best Model
Random Forest achieved the highest accuracy among all models.

## Conclusion
Random Forest outperformed Logistic Regression and KNN because it can capture complex relationships between features and reduce overfitting.
